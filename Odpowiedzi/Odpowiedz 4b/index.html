Zadanie 4. Funkcje Async

W tym zadaniu przybliona zostanie obsługa wyjątków w funkcjach asynchronicznych. Będziemy się posługiwać dwoma rodzajami zapisu operacji asynchronicznych: Promise API oraz async/await.

Celem zadania będzie wyświetlenie wiadomości message przechwyconego błędu. W danych zadania dostarczona jest funkcja: funkcjaZBledem. Jest to funkcja asynchroniczna, które zwraca błąd. W funkcji runAsync należy wykorzystać przechwytywanie błędu odpowiednie dala async/await. W funkcji runPromise należy umieści kod obsługi wyjątków oparty o Promise.

Prawidłowym rozwiazaniem tego zadania jest taki wynik:

string: "Funkcja z błędem"
