Zadanie 4. Funkcje Async

W tym zadaniu przybliżone zostaną pobieranie i wysyłanie danych z/do API. Będziemy się posługiwać wbudowanym w przeglądarkę Fetch API (dokumentację można znaleźć tutaj: Dokumentacja Fetch API). Za źródło danych posłuży nam JSONPlaceholder (Strona o JSONPlaceholder).

Celem zadania będzie wyświetlenie danych pobranych z API oraz stworzenie nowego "postu" i wyświetlenie jego danych. W kodzie umieszczone są API URLe do poszczególnych podpunktów.

Należy przygotować 3 funkcje, które będą wywołane poniższymi przyciskami:

  1. Pobierz 10 postów - należy przygotować funkcję, która pobierze posty i wyświetli id oraz title dziesięciu z nich.
  2. Pobierz 23. post - należy przygotować funkcję, która pobierze post z id 23 i wyświetli jego title oraz body
  3. Stwórz nowy post - należy przygotować funkcję, która wyśle żądanie do API o stworzenie nowego postu z title o wartości SAP JS Course i wyświetli jego ID oraz title
