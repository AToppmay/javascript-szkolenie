Zadanie 3c. Obiekt Object

Mamy podaną tabelę z postaciami, które mają różne własności. Dla każdej postaci wyśietl wszystkie jej własności w parach Klucz: Wartość.

Należy użyć obiektu Object i jego metody .keys().

Rozwiązanie:

Prawidłowym rozwiazaniem tego zadania jest taki wynik: 

string: "imie: Jan"
string: "wiek: 65"
string: "telefon: 795700165"
string: "imie: Ala"
string: "wiek: 27"
string: "zwierze: Kot"
string: "imie: Burek"
string: "wiek: 7"
string: "rasa: Labrador"

Przestrzeń robocza

Ta przestrzeń podłączona jest do funkcji wyświetlającej wyniki kodu. Aby wyświetlić wyniki przetwarzania, należy użyć przygotowanej funkcji "wyswietl".
